专业的风控模型
营普金服风控团队来自银行、基金、小贷、资产管理等传统金融及互联网新金融领域，以传统金融的风控经验结合第三方大数据、人工智能，拥有自己独创的一套风控模型，用于对借款人的情况进行初审。
严谨的风控流程
通过初审的借款人，营普金服会作进一步风控审核，涵盖面审、实物验证、资料收集、信息交叉验证、现场考察、暗访、办理抵质押手续等，确保手续完整，有利于贷后管理。
双重风控 独立审核
每个项目上线发布前，营普金服风控委员会进行二次风控。风控委员会由第三方律师事务所运营负责人、资产端负责人、总经理、风控负责人组成，根据项目风控资料及风控结论，对项目对应的风控经理进行质询。最终结果采用一票否决、全体签字制度。
